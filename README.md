# Button Design System

Famille de composants Button React évolutive, construite autour de couches partagées, de variantes et de design tokens.

## Architecture
BaseButton
├── TextButtonBase
│   ├── Button
│   └── GroupButton
└── IconButton

## Stack
- React
- TypeScript
- SCSS
- Design tokens
